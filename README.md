# Sales Performance Dashboard - Power BI

This project is an interactive sales dashboard created using Power BI.

## KPIs

* Total Revenue
* Total Orders
* Total Units Sold

## Insights

* Electronics category generated the highest revenue
* North region was the top-performing region
* Dashboard includes interactive slicer for region-wise filtering

## Tools Used

* Power BI Desktop
* DAX
* Excel Dataset
* GitHub

## Files Included

* `.pbix` dashboard file
* dataset
